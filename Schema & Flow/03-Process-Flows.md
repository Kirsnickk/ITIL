**States & Transitions:**
- `DRAFT` вҶ’ `PENDING` (Submit request)
- `PENDING` вҶ’ `APPROVED` (All approvals complete)
- `PENDING` вҶ’ `REJECTED` (Any approver rejects)
- `APPROVED` вҶ’ `ORDERED` (PO sent to vendor)
- `ORDERED` вҶ’ `DELIVERED` (Goods received)
- `DELIVERED` вҶ’ `PAID` (Payment completed)

**Notifications:**
- Requestor: Request submitted, approved/rejected, delivered
- Manager: New request pending approval
- Finance: Approved request needs budget review
- IT Staff: Goods delivered, need to create assets
